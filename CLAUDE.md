# 0x7EA · 奇异A纪元 抽奖系统

清华大学软件学院 2026 学生节现场抽奖。主题「奇异A纪元」——2026 的十六进制是 `0x7EA`，设定为在十六进制年份中时空穿梭抽取中奖者。

## 抽奖规则

- **号码池**：公元前 `0x001`（即 -0x200 ~ -0x001）到公元后 `0x200`，不含 0，共 **1024 个**号码。
- **奖项**：
  - 一等奖 · 2 名 → 单抽模式（`draw-single.html`）
  - 二等奖 · 10 名 → 十连抽模式（`draw-ten.html`）
  - 三等奖 · 30 名 → 十连抽模式（跑 3 次）
- **去重**：已中号码永久不再出现（localStorage 持久化）。

## 页面结构

| 路径 | 用途 |
|------|------|
| `index.html` | 主入口，选择抽奖模式 |
| `draw-single.html` | 一等奖：① 抽公元前/后 → ② 抽三位十六进制数字 |
| `draw-ten.html` | 二/三等奖：5×2 十连抽，所有号码同时滚动同时停止 |
| `admin.html` | 查看全部已抽号码 / 导出 CSV / 重置 |

## 现场操作手册

### 快捷键
- **空格** = 开始滚动 / 停止滚动（状态自动切换）
- **B** = 返回主页
- **A** = 进入历史记录（admin）
- **H** = 显示快捷键手册（所有表面按键提示集中在此）
- **R**（仅在 admin 页）= 一键重置，需二次确认

### 一等奖流程（每名 1 次）
1. 主页点「一等奖」
2. 按空格开始，后缀「前 / 后」来回翻滚
3. 再按空格定格 → 点「继续」
4. 按空格开始十六进制滚动
5. 再按空格定格 → 号码自动入库

### 二/三等奖流程（每轮 10 个）
1. 主页点「十连抽」，选择本轮奖项（二等奖 / 三等奖）
2. 按空格，10 个号码框同时开滚
3. 再按空格，所有框从左上到右下依次定格
4. 点「确认并归档」完成本轮

## 故障预案

| 情况 | 处理 |
|------|------|
| 动画卡顿 | 刷新页面，localStorage 已保留历史，不会丢号码 |
| 误点开始 | 再按一次空格即可停止；停止后可在 admin 删掉最后一条记录重抽 |
| 号码重复 | 系统保证不会；如需验证可开 admin 页查看 |
| 需要重来 | admin 页 → 重置（二次确认） |

## 开发说明

- 纯 HTML + CSS + JS，无构建。
- 核心模块：
  - `js/hex.js` — 号码池生成、格式化
  - `js/state.js` — localStorage 状态、去重
  - `js/roller.js` — 滚动动画控制器
  - `js/single.js` / `js/ten.js` — 两种抽奖流程
- localStorage key：`lottery:drawn:v1`
